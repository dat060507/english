# 📚 VocabAI

An AI-powered English vocabulary learning application built with Flutter.

## Features

- 📖 Learn new vocabulary
- 🔁 Review words
- 🎮 Mini Games
- 📊 Statistics
- 🤖 AI Explanation
- 🔊 Pronunciation
- 🧠 Spaced Repetition

## Screens

- Home
- Learn
- Flashcard
- Quiz
- Statistics
- Settings

## Tech Stack

- Flutter
- Dart
- SQLite
- OpenAI API
